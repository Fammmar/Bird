## Название
Bird
<br>

## Описание
Данное мобильное приложение было разработано в рамках учебной практики. 
Приложение позволяет пользователю вести дневник важных задач и получать уведомление чтобы вовремя выполнить важную задачу. 
<br>

## Установка
Для установки программы нужно зайти в раздел `releases` и установить `.apk` из самого нового релиза, 
После нужно найти скачанный файл в своём файловом менеджере и нажать на него. 
В открывшемся окне подтвердить установку 
После завершения процесса окно закроется и приложение будет установлено на устройство.
<br>

## Использование
Мобильное приложение Bird
Создано для предупреждения «пожаров» дедлайнов
Приложение будет напоминать о важных задачах как птичка принося важную весточку
о домашней работе / о сроках сдачи / о оставшемся времени
В нем можно будет распланировать график дня / недели / месяца
и отмечать не только отчет о выполнении но и о стадии выполнения.
<br>
